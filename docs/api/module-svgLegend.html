legendDefintion

Type:
  • Object
Properties:
Name Type Description
title string a title caption for the legend.
size number a number between 0 and 1. This is one factor taken into account to calculate the size of the legend dots.
precision number should match the precision (in metres) of the map dot, e.g. 2000 for tetrads. This is one factor taken into account to calculate the size of the legend dots.
opacity number a number between 0 and 1 indicating the opacity of the legend symbol. 0 is completely transparent and 1 is completely opaque.
lines Array.<legendLine> an arry of objects representing lines in a legend.

legendLine

Type:
  • Object
Properties:
Name Type Description
color string a colour for the legend symbol which can be hex format, e.g. #FFA500, RGB format, e.g. rgb(100, 255, 0) or a named colour, e.g. red.

text string specifies the text for the legend line.
